# Personal Portfolio v2

A modern portfolio website built with Next.js, TypeScript, and Supabase.

### Development

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Technologies Used
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
